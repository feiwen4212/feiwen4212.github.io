    <h2 id="dynamic-programming-%e5%8a%a8%e6%80%81%e8%a7%84%e5%88%92">Dynamic Programming 动态规划</h2>
<blockquote>
<p>The key idwa of DP, and of reinforcement learning generally, is the use of value functions to organize and structure the search for good policies. - Sutton</p>
</blockquote>

delta = 0
loop for each s ∈ S:
    v = 0
    for a ∈ A:
        for s' ∈ S:
            v += π(a|s)[r(s,a) + γ * P(s'|s,a) * V(s')]
    delta = max(delta, abs(v-V(s)))
    V(s) = v
until delta &lt; θ
